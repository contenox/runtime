# Support

Use GitHub issues for Contenox VS Code extension bugs, rough edges, questions,
and feature requests:

https://github.com/contenox/runtime/issues

Useful details include:

- VS Code version
- Contenox extension version
- operating system or remote workspace environment
- configured provider/model
- relevant output from `Contenox: Show Output`
- relevant JSONL events from `Contenox: Show Telemetry Log`

Do not report security vulnerabilities in public issues. See
[SECURITY.md](SECURITY.md).

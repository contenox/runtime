# Security

Contenox can run a local runtime, inspect workspace files, call configured
providers, and request tool execution through explicit approval flows. Please do
not report security vulnerabilities in public issues.

Report vulnerabilities by email:

hello@contenox.com

Include a short description, reproduction steps, affected version, operating
system, and any relevant workspace or chain configuration. Vulnerabilities may
include unsafe local tool behavior, approval bypasses, credential exposure,
workspace trust bypasses, or execution outside intended policy boundaries.

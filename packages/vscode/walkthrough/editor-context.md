# Editor Context

Contenox can attach the current editor, selected text, diagnostics, and git
changes to a request.

Common entry points:

- Select code, then run `Contenox: Ask About Selection`
- Run `Contenox: Explain Diagnostics`
- Run `Contenox: Review Workspace Changes`
- Run `Contenox: Draft Commit Message`

Tool and file operations still pass through Contenox runtime policy and HITL
approval.
